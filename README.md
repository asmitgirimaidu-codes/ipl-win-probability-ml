# ipl-win-probability-ml
A machine learning pipeline designed to quantify the unpredictability of T20 cricket. By analyzing factors like remaining runs, wickets in hand, current run rate, and historical venue bias, this project generates a live, evolving "win probability percentage" for IPL matches. No guessing, just pure data.
## ✨ Features
* **Real-Time Predictions:** Calculates winning probability ball-by-ball.
* **Context-Aware ML:** Accounts for venue history, target score pressure, and the "death overs" effect.
* **Interactive Dashboard:** Beautiful visualizations showing the swing of momentum throughout the match.
